# IlloFlex UI

IlloFlex UI is the frontend application of IlloFlex, a vocational guidance platform focused on Andalusia.

It allows users to complete the anonymous vocational test, review their RIASEC result, explore recommendations, and recover saved results through a temporary link.

## Main Features

- Landing page
- Anonymous test flow
- Step-by-step question navigation
- Result visualization
- Recommendation display
- Temporary result link generation
- Public result recovery by token

## Tech Stack

- React
- Vite
- React Router
- Axios
- Material UI

## Project Structure

- `pages` for main screens
- `components` for reusable UI
- `services` for API calls
- `router` for route definitions

## Dependencies and Local Setup

Install dependencies:

```bash
npm install
```

Run in development mode:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
